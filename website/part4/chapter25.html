Chapter 25. Shell, Text Editor, and Compiler

You can follow the instructions in this chapter without doing those of the previous ones. For this, before doing them:

  • with the Arduino:
    • assemble the toy computer components as explained in the assembly instructions.
    • connect the Arduino to your computer. Press the ERASE button for a few seconds, then press the RESET button. In a terminal on the host computer, type python3 flash_helper.py < backups/shell.txt (in the directory where you unzipped scripts.zip).
  • with the emulator: open this page.
